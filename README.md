## 项目简介
这是一个以“完美”为目标的游戏服务端项目。  

## 功能

- 商店系统：可正常浏览与购买。
- 编队系统：角色上阵、换位、切队流程完整。
- 卡池抽卡：支持完整抽卡流程与结果展示。
- 战斗流程：可进入战斗并完成主要战斗环节。
- 大世界场景：场景加载、交互、基础探索可用。
- 角色养成：升级、晋阶等核心养成链路可用。
- 任务系统：主线与常见任务流程可推进。
- 好友系统：基础好友交互与展示功能可用。
- 异相仲裁：支持入口解锁、关卡挑战、阶段奖励领取等核心流程。
- 挑战玩法：忘却之庭 / 虚构叙事 / 末日幻影可进入并游玩。
- 货币战争：已实现基础玩法流程。
- 成就系统：大部分成就可追踪与完成。
