## 專案簡介
這是一個以「完美」為目標的遊戲服務端專案。

## 功能

- 商店系統：可正常瀏覽與購買。
- 編隊系統：角色上陣、換位、切隊流程完整。
- 卡池抽卡：支援完整抽卡流程與結果展示。
- 戰鬥流程：可進入戰鬥並完成主要戰鬥環節。
- 大世界場景：場景載入、互動、基礎探索可用。
- 角色養成：升級、晉階等核心養成鏈路可用。
- 任務系統：主線與常見任務流程可推進。
- 好友系統：基礎好友互動與展示功能可用。
- 異相仲裁：支援入口解鎖、關卡挑戰、階段獎勵領取等核心流程。
- 挑戰玩法：忘卻之庭 / 虛構敘事 / 末日幻影可進入並遊玩。
- 貨幣戰爭：已實現基礎玩法流程。
- 成就系統：大部分成就可追蹤與完成。
